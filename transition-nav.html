Navigation Styling

This nav has two states. First is the default state that appears when the page hasn't scrolled. Once the page is scrolled more than 100px, the nav switches to its scroll state.

1. Style this nav to have a larger default state with a compact scrolled variation.

2. Give the user a clear visual distinction between the two states.

3. Experiment with using the _transition snippet to apply various transitions to the nav.

The css/transition-nav.css file has the css you need for this exercise.
