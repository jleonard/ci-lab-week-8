Use this example to experiment with css animations.

In css/animations.css you'll find css to work with. There are rulesets for the box's inactive and hover states.

First you need to add some rules to the :hover state to change the box's styles.

some css properties to experiment with...


opacity: 0.5;
color: green;
background-color: #c30;
font-size: 32px;
transform: scale(0.8,0.8);
transform: skew(15deg,15deg);
transform: rotate(15deg);

Next, add a css transition. Experiment with changing the various properties to create different transition styles.


transition-property: all;
transition-duration: 750ms;
transition-timing-function: ease-in;
transition-delay: 500ms;
