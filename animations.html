Use this example to experiment with css animations.

In css/animations.css you'll find css to work with. There are rulesets for the box's active (animating) and default states.

Work with the .css in css/animations.css.

Part 01, create your own keyframe animation

At the bottom of the file you'll find some example @keyframe animations to experiment with.

An example keyframe animation looks like this...


@-webkit-keyframes wiggle {
  0% {
    transform: rotate(20deg);
  }
  100% {
    transform: rotate(-20deg);
  }
}

Use these examples as a guide to create your own.

Part 02, apply that animation to the box.

Use the _animation snippet to apply the css animation properties to the box's css. Toggle the apply button to toggle the animation on and off.

The css animation snippet output will look like this...



/*
 * Name specifies the keyframe animation to apply
 */ 
-webkit-animation-name : keyframes;
        animation-name : keyframes;

/*
 * Duration specifies the length of time it
 * takes to complete a single animation cycle.
 * Options are seconds or milliseconds (ex. 1.5s or 1500ms)
 */     
-webkit-animation-duration : 0.5s;
        animation-duration : 0.5s;

/*
 * The number of times an animation cycle will repeat before stopping.
 * Options are positive whole numbers, decimals and infinite.
 */
-webkit-animation-iteration-count : 1;
        animation-iteration-count : 1;

/*
 * The delay before the animation starts.
 * Options are seconds or milliseconds (ex. 1.5s or 1500ms)
 */
-webkit-animation-delay : 1.5s;
        animation-delay : 1.5s;

/*
 * Direction indicates whether the animation should play 
 * in reverse on alternate cycles.
 * Options are normal, reverse, alternate, alternate-reverse
 */
-webkit-animation-direction : reverse;
        animation-direction : reverse;

/*
 * The timing function defines the style of progression
 * Options are documented here https://developer.mozilla.org/en-US/docs/Web/CSS/animation-timing-function
 */
-webkit-animation-timing-function : linear;
        animation-timing-function : linear;

/*
 * Plays or pauses an animation
 * Options are running or paused
 */
-webkit-animation-play-state : running;
        animation-play-state : running;
